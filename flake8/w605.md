# Invalid escape sequence 'x' (W605)

As of Python 3.6, a backslash-character pair that is not a valid escape sequence now generates a `DeprecationWarning`. This will eventually become a `SyntaxError`.

### Anti-pattern

```python
regex = '\.png$'
```

### Best practice

```python
regex = r'\.png$'
```
